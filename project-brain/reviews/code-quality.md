# Code Quality Rubric
- Architecture (15%): Modularity, separation of concerns.
- Maintainability (15%): Code readability, docstrings.
- Reproducibility (15%): Static seeds, configuration-driven logic.
- Data Leakage (15%): No train/test overlap.
- Security (10%): No exposed API keys/tokens.
- Scalability (10%): Model execution speed, efficient vectors.
- Side Effects (10%): No unexpected file modifications.
- Overall (10%): Final validation.
